特征工程
A. 赔率特征
初始赔率特征：主胜、平局、客胜的初始赔率
赔率变化序列：构建时间序列矩阵（N×3×T，T为时间步）
衍生特征：
-赔率离散度：(max_odds - min_odds) / avg_odds
赔付率计算：1/(1/主胜+1/平局+1/客胜)
凯利指数计算：
- 主胜：(主胜赔率 - 1) / 主胜赔率
- 平局：(平局赔率 - 1) / 平局赔率
- 客胜：(客胜赔率 - 1) / 客胜赔率
赔率变化率（临场vs初赔）
- 主胜：(临场主胜赔率 - 初赔主胜赔率) / 初赔主胜赔率
- 平局：(临场平局赔率 - 初赔平局赔率) / 初赔平局赔率
- 客胜：(临场客胜赔率 - 初赔客胜赔率) / 初赔客胜赔率
